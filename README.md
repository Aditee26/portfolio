## About

This portfolio showcases my projects, skills, and certifications through a retro pixel-art interface. Rather than reaching for a framework, I built it entirely with vanilla HTML, CSS, and JavaScript to demonstrate a solid grasp of core front-end fundamentals — DOM manipulation, responsive layout, and custom-built interactive components — while keeping the site fast and dependency-free.

---

## Features

- Pixel-themed UI with a custom animated loading screen
- Pixel-style navigation system
- Interactive, hover-animated project cards
- Custom-built certification carousel (no library)
- Live LeetCode stats section via API integration
- Fully responsive across desktop, tablet, and mobile

---

## Tech Stack

| Category | Details |
|---|---|
| **Frontend** | HTML5, CSS3, Vanilla JavaScript |
| **Layout & Styling** | Flexbox, CSS Grid, custom animations, CSS variables for theming |
| **Interactivity** | DOM manipulation, custom carousel logic, scroll/loading animations |
| **Integrations** | LeetCode public API |
| **Deployment** | Netlify |

## Project Structure

```
portfolio/
├── index.html
├── styles/
│   └── ...           stylesheets
├── scripts/
│   └── ...           JavaScript modules
└── assets/
    ├── profile.png
    ├── certificates/
    └── project-images/
```

> The current version runs primarily through a single `index.html` for simplicity — structure will modularize further as the site grows.

---

## Running Locally

```bash
git clone https://github.com/Aditee26/portfolio.git
cd portfolio
```

Then open `index.html` directly in your browser — no build step or dependencies required.

---

## Customizing

This portfolio is built to be easily forked and adapted:

1. Replace personal information in `index.html`
2. Swap in your own project cards
3. Update the certification carousel content
4. Change the color theme via CSS variables
5. Replace the profile image and project screenshots in `assets/`
